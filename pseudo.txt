Example 1: An algorithm that asks a user to enter a positive number repeatedly until the user enters a zero value, then determines and outputs the largest of the numbers that were entered.

set largest to zero
request a positive number from the user
while number is not equal to zero
    if number is greater than largest
        set largest to number
    end if
    request another positive number from the user
end while
if largest is not equal to zero
    print "The largest number is: ", largest
else
    print "No positive numbers were entered"
end if
